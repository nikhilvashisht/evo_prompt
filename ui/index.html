SDK Interceptors (Recommended)

Intercept requests at the application layer with our simple client wrappers. Instantly records high-fidelity trace objects directly to the DB.

from openai import AsyncOpenAI
from app.interceptors import OpenAIInterceptor

client = AsyncOpenAI()
# Wraps original client asynchronously
OpenAIInterceptor(client)

# That's it! All client completions are automatically logged
response = await client.chat.completions.create(...)
